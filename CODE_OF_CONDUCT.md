# Code of Conduct

We want this project to be a safe, welcoming, and productive environment.

## Our standards
Examples of behavior that contributes to a positive environment:
- being respectful and considerate
- giving and receiving constructive feedback
- focusing on what’s best for the project and users
- assuming good intent, asking clarifying questions

Unacceptable behavior includes:
- harassment or discrimination
- insults, personal attacks, or trolling
- publishing private information without permission
- advocating violence or hate

## Reporting and enforcement
If you experience or witness unacceptable behavior, report it to the maintainers.

Maintainers may:
- request changes to behavior
- remove content
- temporarily or permanently ban contributors

## Scope
This code of conduct applies within all project spaces, including issues, PRs, and chat channels used for development.
