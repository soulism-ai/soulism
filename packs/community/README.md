# Community Packs

This folder is reserved for community-contributed persona packs.

Rules:
- must pass schema validation
- must be signed
- must include clear safety boundaries

## Community Pack

- `manifest.json`: pack metadata, compatibility, and provenance block
- `personas/*.yml`: composable persona definitions
- `signatures/pack.sig.json`: pack signature metadata
- `signatures/public_key.pem`: verification key for signed metadata
